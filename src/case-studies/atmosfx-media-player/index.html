Project Overview

AtmosFX needed a modern, accessible media player interface that would enhance user engagement and make content discovery more intuitive for their diverse user base.

The existing media player was functional but lacked modern design patterns and accessibility features. Users reported difficulty navigating content and accessing player controls, particularly on mobile devices.

Project Details

  • Role: Lead UX Designer
  • Duration: 8 weeks
  • Team: 3 designers, 2 developers
  • Tools: Figma, Axure, Sketch
  • Platform: Web & Mobile

The Problem

User Pain Points

  • Difficult content navigation and discovery
  • Inconsistent player controls across devices
  • Poor accessibility for screen reader users
  • Outdated visual design that didn't match brand
  • Limited mobile optimization

Business Goals

  • Increase user engagement and session duration
  • Improve content discovery and consumption
  • Enhance brand perception with modern design
  • Ensure accessibility compliance (WCAG 2.1 AA)
  • Optimize for mobile-first experience

Approach & Process

Research

Conducted user interviews, usability testing, and competitive analysis to understand user needs and industry best practices.

Design

Created wireframes, prototypes, and high-fidelity designs focusing on accessibility and modern UX patterns.

Implementation

Collaborated with development team to ensure design fidelity and accessibility compliance throughout build.

Outcomes & Impact

User Experience Improvements

  • 40% increase in content discovery
  • 25% improvement in session duration
  • 90% accessibility compliance score
  • 60% reduction in support tickets

Key Features Delivered

  • Intuitive content navigation with smart recommendations
  • Consistent player controls across all devices
  • Full keyboard navigation and screen reader support
  • Modern, responsive design system
  • Enhanced mobile experience with touch-optimized controls
